# Unicode Hovery

Display a hovering tooltip on / underline under escaped unicode/hexadecimal sequences.

## Features

- Supports multiple escaped text formats (e.g. `\u{1F44D}`, `\x70`, `\u3042`, `\uD83D\uDE0D`, `U+1F44D`)
- Customizable underline options
